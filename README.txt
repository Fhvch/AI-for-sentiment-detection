Данная программа с нейросетью анализирует тональность отзыва.

app.py содержит интерфейс для удобства работы с нейросетью

В папке src:
            config.py - это настройки обучения нейросети
            train.py - это обучение нейросети
            data_preprocessing.py - это обработка данных для обучения в train.py
            predict.py - это пропуск через нейросеть данных введенных в интерфейсе
            model.py - загрузка модели для обучения в train.py

В папке data:
             В папке models лежит обученная модель
             В папке raw лежит файл reviews.csv с набором данных для обучения нейросети

Запустить программу можно через терминал командой python app.py

Запустить обучение нейросети можно через терминал командой python src/train.py

Зависимости в файле requirements.txt
